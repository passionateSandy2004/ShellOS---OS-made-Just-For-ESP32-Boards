ShellOS

A real shell, filesystem, packages, and WiFi — on a microcontroller. Flash boards from Windows with one download.

Install for Windows

Downloads ShellOSImager.exe — run it, pick your COM port, and flash. No separate Python install.

What is ShellOS?

ShellOS is built on ESP-IDF and FreeRTOS. You get a serial shell, file commands, WiFi, loadable packages, Lua, and tools like a TCP remote shell — ideas from Unix, scaled for ESP32-class hardware.

  • Interactive shell with filesystem and network commands
  • Packages you can run, stop, and remove from the device
  • Windows imager with firmware for ESP32-CAM and ESP32-C6 targets
